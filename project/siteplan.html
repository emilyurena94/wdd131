Site Plan — Blackpink Fan Page

Site Name

Blackpink Fan Page

The name clearly identifies the topic and purpose, making it easy for fans and new visitors to recognize the site.

Site Purpose

To provide a fun, organized, and informative place about the band Blackpink, including biography, discography, gallery, and interactive content (fun facts and song recommendations).

Audience

  • Fans of Blackpink
  • People interested in K-pop and music in general
  • Visitors looking for albums, songs, or achievements

Scenarios

  1. "I want to learn basic information about Blackpink and their history."
  2. "Where can I find albums, release dates, and top songs?"
  3. "I want to see photos of the band."
  4. "I want to see a random fun fact or song recommendation."

Pages to Include

  1. Home (introducción, hero image, enlaces rápidos)
  2. Biography (Early life, career, achievements)
  3. Discography (Album covers, release dates, top songs)
  4. Gallery (Image gallery)
  5. Fun Facts (Interactive JS section with random facts or song recommendations)

Color Scheme

  • Primary — Pink Usage: headings, buttons, highlighted links.
  • Secondary — Black Usage: main text, borders, icons.
  • Background — White Usage: page background, cards, readability.

This document uses the same colors to demonstrate consistency.

Typography

  • Poppins — headings for a modern, bold look
  • Roboto — body text for readability on screens

This document uses the same fonts to show intended usage.

Wireframes (mobile and desktop) — descriptions only

Mobile: Vertical layout: header , hero image, intro text, stacked quick links, featured section, footer (socials & copyright).

Desktop: Horizontal navigation, hero banner, two-column content, featured section, footer.

Naming Conventions

All files and folders will follow course conventions: lowercase letters, hyphens instead of spaces (discography.html, styles/main.css, images/blackpink-2024.jpg).
